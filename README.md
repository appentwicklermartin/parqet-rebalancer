# Parqet Rebalancing Dashboard

Persönliches Portfolio-Rebalancing-Dashboard mit Parqet-Integration.

## Setup

1. Repository auf Render.com oder Railway.app deployen
2. Environment Variables setzen:
   - `PARQET_CLIENT_ID`: Deine Parqet Client ID
   - `BASE_URL`: Die URL deiner App (z.B. https://deine-app.onrender.com)
3. In Parqet Developer Hub als Integration eintragen

## Lokal starten

```bash
npm install
node server.js
```
